Ad Only Plugin

This plugin allows you to play only IMA3 ads in a player. No normal video content is supplied.

The sample video below demonstrates the use of the Ad Only Plugin. The ad that plays is configured as a preroll in a VMAP/VAST XML document. In this case the player is NOT set to autoplay, which you would most likely want for an ad only player, but that is a setting you can change.

Implement using Players module

To implement the Ad Only Plugin using the Players module, follow these steps:

  1. In Studio, open the PLAYERS module and create a player, or locate a player to which you want to add the plugin.
  2. Click the link for the player to open the player properties.
  3. Locate the ADVERTISING section and click the EDIT button, and click Yes to enable IMA ads.
  4. Supply a Server URL. For testing purposes you can use the following VMAP/VAST XML document that will display a single pre-roll ad. 
    //solutions.brightcove.com/bcls/brightcove-player/vmap/one-preroll-vmap.xml
  5. Change the Request Ads behavior to On demand.
  6. Check to be sure your ADVERTISING section appears as follows, but using your server URL.
  7. Save the advertising information.
  8. Locate the Plugins section and click Edit. You will be entering one JavaScript URL for this plugin.
  9. The JavaScript for the plugin is provided and will not change for different player implementations. For the JavaScript URL, enter: 
    //players.brightcove.net/videojs-ima3-tech/1/videojs-ima3-tech.min.js
  10. Check to be sure your PLUGINS section appears as follows.
  11. Click Save and then Publish the player.

You can now use either the iframe or in-page embed player implementations to use the ad only player.
