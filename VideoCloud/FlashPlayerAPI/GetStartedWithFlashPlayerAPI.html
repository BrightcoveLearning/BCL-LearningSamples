Tip of the week: Player API basics:

Getting information about the current video

There are all kinds of reasons why you might want to retrieve information about the video currently in the player. Here's a simple example that shows you how easy it is to do that using the Player API -- here I'm just getting the thumbnail URL and display the image and URL under the player. I'm using the getCurrentVideo() method of the VideoPlayer Module, and set up a listener for MediaChange events, so that I can make a new call to the method when a new video is loaded. You can get this sample at http://files.brightcove.com/BCLGetThumbnailURL.html
