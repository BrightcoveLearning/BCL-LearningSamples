Steps for Implementing CTA Videos

  1. Store the image you want to use in PNG format with a transparent background on http://img.brightcove.com/
  2. Save the data for the image/click-thru: In the Long Description field for the video store a JSON string in like this: 
    {
  "linkURL" : "http://zencoder.com",
  "imageURL" : "http://img.brightcove.com/BCL-cta-sample.png",
  "top" : "160px",
  "left" : "480px"
}

    Note that the white-space doesn't just matter, but all the syntax does — there is a free JSON tool that you can create / open this code in that will check the syntax and make sure it is correct.

    The top and left values are pixel distance from the top and left edges of the video player to the top and left edges of the image.

  3. Set the code cue points for the positions where the CTA should appear and disappear: set two cue points with the following names:
    • First cue point name: ctaStart
    • Second cue point name: ctaEnd

    Note that the names are case-sensitive! Also if it would easier for you to specify the start and end times in second (e.g. 14.33) in the long description, I can modify the code to work that way.

  4. Embed the embed-code-for-cta-player.txt code block (as is). There is one number to change:
    • The video ID (value of the @videoPlayer param of the object tag)
