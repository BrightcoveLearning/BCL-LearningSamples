Fixed Player Sample

For fixing the position of a player on the page, you need to think outside box — specifically, the player box. The player is just another element in the HTML, and there for you can manipulate its appearance, including its size and position, using CSS.

An important thing to remember is that you probably will not get the results you want if you try to fix just the video player itself, with other content flowing around it. Fixing content on a page really only works for a whole column of content.

The column can be on the right or left side, and you can fix it completely, so that it will not move at all, or you can let it be scrolled independently of the other column(s).

Here the video is at the top of a fixed column on the left side. The solution here uses CSS - no JavaScript required.

The key is the CSS position property, which can have values of:

  • static — normal positioning, so this does nothing
  • relative — a block with this positioning will behave exactly like a static block, but positioning it relatively allows you to nest other blocks with absolute or fixed positioning
  • abolute — this positions the block absolutely based on distances from the top, left, bottom, and/or right boundaries of the block that contains this one, its parent
  • fixed — a fixed block will not move at all

The key style classes here, then, are as follows:


  /* the overall content wrapper,
  positioned relatively,
  to allow other positionings inside it */
  .BCLbodyContent {
    margin-left: 20px;
    margin-right: 20px;
    position: relative;
  }
  /* the main content, positioned absolutely,
  to shift it to the right, 
  but it's allowed to scroll */
  .BCLscrollingContent {
    position: absolute;
    left: 340px;
    margin-left: 20px;
  }
  /* the video player column, with fixed
  postioning, so that it remains in
  view even when the viewer scrolls down */
  .BCLfixedContent {
    position: fixed;
    left: 20px;
    top: 70px;
    width: 320px;
  }
