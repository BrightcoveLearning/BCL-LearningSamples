Brightcove Learning Sample: Overlaying Other Objects on Top of a Player

I've had several people ask me how to overlay an HTML object like an image or button over a Brightcove Video Cloud player. You can easily do this using CSS positioning.

You can see an example below.

To do this, simply place both the player publishing code and the object you want to place on top of it in the same <div> tag. Give your HTML object a class or id so that you can style it. Create CSS style rules for both the BrightcoveExperience (the player object) class, and the class or id for your other object.

In each style rule include the position property set to "absolute", and right, left, top, and/or bottom properties to position the object horizontally and vertically with the containing div tag. Also include the z-index property, making sure that the value for your HTML object is higher than the value for the player object.

Here's sample code for the player and button you see above.

      <style>
      .BrightcoveExperience {
      	position: absolute;
      	top: 0px;
      	left: 0px;
      	z-index: 1;
      }
      .overlay {
      	position: absolute;
      	top: 60px;
      	left: 20px;
      	z-index: 5;
      }
      .playContainer {
      	position: static;
      }
      </style>
      <div class="playContainer">
      <!-- Start of Brightcove Player -->
<button class="overlay">My Overlaid Button</button>

</div>
