Creating a Player with a Transparent Background: Would you like your video player to have a transparent background? You can do this in BEML by adding a backgroundColor attribute to the layout tag: 

<Layout backgroundColor="0x01000000">

The 01 value bolded above is the alpha channel set to full transparency (a value of 00 would give you full opacity). You will also need to add a parameter to the JavaScript publishing code:

<param name="wmode" value="transparent" />


If you have an Express account and can't use BEML, you can set the background color for the player through the Player API -- note that in all the built-in player templates, the <Layout> tag has the id "application," which allows you to get a reference to it using the getElementByID() method.
