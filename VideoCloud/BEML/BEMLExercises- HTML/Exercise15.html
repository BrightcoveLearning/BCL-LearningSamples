Exercise: Bind playlist data in a template

In this exercise, you will learn how to bind playlist data to a video player template.

Examine playlist information

  1. Login to your Brightcove Video Cloud account.
  2. Navigate to the Media module.
  3. Select All Playlists and locate the playlist you created in Exercise 1. If you have not created a playlist, refer to Exercise 1 on how to create a playlist.
  4. With your playlist selected, click Open.
  5. Select the Edit Settings link at the top of the playlist page.
  6. In the Edit Playlist dialog, select Show additional information. You can see that you can define a name, description, and reference ID for your playlist.
  7. Browse to the the Flash-Only Player API documentation on the Brightcove Support site:
    • http://docs.brightcove.com/en/player/
  8. In the All Classes, select the MediaCollectionDTO class.
  9. Review the Public Properties section. You should see properties that correspond to all of the playlist attributes, including displayName, shortDescription, and referenceID.
  10. Return to Video Cloud Studio.
  11. If you have not done so already, in the Edit Playlist dialog, fill out the Name and Description fields.
  12. Select Save Playlist.

Bind playlist data to a video player template

  1. Navigate to the Publishing module.
  2. Select All Templates and locate the BEMLListComponents template.
  3. Select Edit.
  4. Locate the TileList component, and add the data attribute with a value that binds the playlistTabs.selectedItem.videoDTOs property.
   
<TileList id="videoList"
   columnWidth="250" rowHeight="90"
   data="{playlistTabs.selectedItem.videoDTOs}">
  1. Select Save Changes.
  2. If you see a warning dialog that states, These changes may affect live players, select Save. Remember that you created a player from this template in a previous exercise.
  3. Select New Player.
  4. In the Create New Player dialog, name your player BEMLListComponentsPlayer and select the BEMLListComponents template.
  5. Select Create Player.
  6. Navigate to the Media module.
  7. Select All Players and locate the BEMLListComponentsPlayer.
  8. Select Add Playlists.
  9. In the Add Playlists dialog, select the playlist you created.
  10. Select Save & Close.
  11. Select Preview. You should see that the tab displays your playlist's name. However, notice that no real video data is displayed. In the next exercise, you will learn how to bind data for a list's selected item.
