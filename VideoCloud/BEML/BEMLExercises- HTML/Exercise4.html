Exercise: Use layout containers

In this exercise, you will learn how to use the different BEML layout containers.

Use the Canvas container

  1. Login to your Brightcove Video Cloud account.
  2. Navigate to the Publishing module.
  3. Select New Template.
  4. In the Create New Template dialog, name the template BEMLLayoutContainers.
  5. To the opening Layout tag, add a width property with a value of 486 and a height property with a value of 600.
           
<Layout id="application"
   width="486" height="600">
   ...
</Layout>
  1. Within the Layout tag block, add an Image component and assign the component the following properties and values:
    • x: 30
    • y: 50
    • width: 223
    • height: 54
    • source: http://img.brightcove.com/brightcove-logo.png
          
<Layout ...>
   <Image x="30" y="50"
   	 width="223" height="54"
       source="http://img.brightcove.com/brightcove-logo.png"/>
</Layout>
  1. Select Preview Template. You should see the image displayed in the preview window.
  2. Select Close.
  3. Copy the Image component and paste two instances of the component below the first.
  4. To the second instance, change the y property value to 150.
  5. To the third instance, change the y property value to 250.
  6. Select Preview Template. You should see three images displayed in the preview window, stacked vertically.
  7. Select Close.
  8. Wrap the Image components within a Canvas container.
          
<Layout ...>
   <Canvas>
       <Image .../>
       <Image .../>
       <Image .../>
   </Canvas>
</Layout>
  1. Preview the template. You should see that the layout of the images has not changed because the Canvas layout container is the default layout.
  2. Select Close.

Use the VBox container

  1. Change the Canvas container to a VBox container.
          
<Layout ...>
   <VBox>
       <Image .../>
       <Image .../>
       <Image .../>
   </VBox>
</Layout>
  1. Preview the template. You should see that the images are still stacked vertically, but their x and y values have been ignored and there is no space between the components.
  2. Select Close.
  3. To the opening VBox container tag, add the gutter property with a value of 30.
          
<VBox gutter="30">
	...
</VBox>
  1. Preview the template. You should see that there is now a space between each of the images.
  2. Select Close.

Use the HBox container

  1. Change the VBox container to a HBox container.
         
<Layout ...>
   <HBox>
       <Image .../>
       <Image .../>
       <Image .../>
   </HBox>
</Layout>
  1. Preview the template. You should see that the images are now laid out horizontally.
  2. Select Close.

Use the boxType attribute

  1. Remove the HBox container tags.
  2. To the opening Layout tag, add the boxType attribute and set its value to vbox.
          
<Layout id="application"
	...
   boxType="vbox">
  1. Preview the template. You should see that the images are stacked vertically.
  2. Select Close.
  3. Change the value of the boxType attribute to hbox.
         
<Layout id="application"
	...
   boxType="hbox">
  1. Preview the template. You should see the images laid out horizontally.
  2. Select Close.
