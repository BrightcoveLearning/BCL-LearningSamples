Exercise: Layout basic components

In this exercise, you will learn more about basic BEML components, including Spacer, Label, TitleLabel, Link, and Image.

Create a new video template

  1. Login to your Brightcove Video Cloud account.
  2. Navigate to the Publishing module.
  3. Select New Template.
  4. In the Create New Template dialog, name the template BasicBEMLComponents and paste the following starter code within the editor:
          
<Runtime>
   <Theme name="Deluxe" style="Light"/>
   <Layout id="application"
        boxType="vbox"
        width="640" height="700"
        padding="3">
        
   </Layout>
</Runtime>

Use basic BEML components

  1. Within the Layout container, add an Image component with a width of 480, a height of 80, and a source property of http://img.brightcove.com/brightcove-logo.png.
          
<Layout id="application" ...>
   <Image width="480" height="80"
       source="http://img.brightcove.com/brightcove-logo.png"/>
</Layout>
  1. Below the Image component, add a ChromelessVideoPlayer component with an id property of videoPlayer, a width property of 640, and a height of 450.
          
<Layout id="application" ...>
   <Image width="480" height="80"
       source="http://img.brightcove.com/brightcove-logo.png"/>
   <ChromelessVideoPlayer id="videoPlayer"
       width="640" height="450"/>
</Layout>
  1. Preview the template. You should see the image and video player laid out vertically. Notice that there is no space between the two components.
  2. Select Close.
  3. Between the Image and ChromelessVideoPlayer components, add a Spacer component with height property set to 10.
          
<Layout id="application" ...>
   <Image width="480" height="80"
       source="http://img.brightcove.com/brightcove-logo.png"/>
   <Spacer height="10"/>
   <ChromelessVideoPlayer id="videoPlayer"
       width="640" height="450"/>
</Layout>
  1. Preview the template. You should now see a space between the image and video player components.
  2. Select Close.
  3. Below the ChromelessVideoPlayer component, add a Spacer component with a height property set to 10.
  4. Below the ChromelessVideoPlayer component, add a TitleLabel element with a text property set to My Video Title, a height property of 40, and a size property of 24.
          
<Layout id="application" ...>
   <Image width="480" height="80"
       source="http://img.brightcove.com/brightcove-logo.png"/>
   <Spacer height="10"/>
   <ChromelessVideoPlayer id="videoPlayer"
       width="640" height="450"/>
   <Spacer height="10"/>
   <TitleLabel text="My Video Title"
       height="40"
       size="24"/>
</Layout>
  1. Below the TitleLabel component, add a Label component with a text property set to My video description, a height property of 20, and a size property of 14.
          
<Layout id="application" ...>
   <Image width="480" height="80"
       source="http://img.brightcove.com/brightcove-logo.png"/>
   <Spacer height="10"/>
   <ChromelessVideoPlayer id="videoPlayer"
       width="640" height="450"/>
   <Spacer height="10"/>
   <TitleLabel text="My Video Title"
       height="40"
       size="24"/>
   <Label text="My video description"
       height="20"
       size="14"/>
</Layout>
  1. Below the Label component, add a Link element with a text property of My Link, a url property of www.brightcove.com, a height property of 20 and a size property of 14.
          
<Layout id="application" ...>
   <Image width="480" height="80"
       source="http://img.brightcove.com/brightcove-logo.png"/>
   <Spacer height="10"/>
   <ChromelessVideoPlayer id="videoPlayer"
       width="640" height="450"/>
   <Spacer height="10"/>
   <TitleLabel text="My Video Title"
       height="40"
       size="24"/>
   <Label text="My video description"
       height="20"
       size="14"/>
   <Link text="My Link"
       url="www.brightcove.com"
       height="20"
       size="14"/>
</Layout>
  1. Select Create Template.
  2. Select New Player.
  3. In the Create New Player dialog, name the player BasicBEMLComponentsPlayer and select BasicBEMLComponents as the template.
  4. Select Create Player.
  5. With the BasicBEMLComponentsPlayer video player select, click Preview to preview the player in the browser. You should see the TitleLabel, Label, and Link components displayed below the video player. When you click the link, you should be directed to the Brightcove homepage.
