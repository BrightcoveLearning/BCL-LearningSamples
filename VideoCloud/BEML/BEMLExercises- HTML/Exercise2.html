Exercise: Create a video player from a template

In this exercise, you will learn how to create a simple BEML template and then create a video player from that template.

Create a BEML template

  1. Login to your Brightcove Video Cloud account.
  2. Navigate to the Publishing module.
  3. In the Publishing module, select New Template.
  4. In the Create New Template dialog, name your template SimpleBEMLTemplate and locate the editor window.
  5. To the Layout element, add the width property with a value of 486 and the height property with a value of 416.
          
<Layout id="application"
   width="486" height="416">
   <!--Insert layout here-->
</Layout>
  1. Within the Layout tag block, add an instance of the VideoPlayer component with an id attribute of videoPlayer.
          
<Layout id="application"
   width="486" height="416">
   <VideoPlayer id="videoPlayer"/>
</Layout>
  1. Your code should look as follows:
          
<Runtime>
	<Layout id="application"
  	 width="486" height="416">
  	 <VideoPlayer id="videoPlayer"/>
	</Layout>
</Runtime>
  1. Select Validate Code to verify that your code is correct.
  2. Click Preview Template.
  3. In the preview window, click Close.
  4. In the Create New Template dialog, click Create Template.

Create a video player from a template

  1. In the Publishing module, click New Player.
  2. In the Create New Player dialog, name the player SimpleBEMLPlayer and select SimpleBEMLTemplate as the template.
  3. Select Create Player.

Note: If you try to add a playlist to the player, your videos will not play because you have not bound any data to the components. You will learn how to bind data in a later exercise.
