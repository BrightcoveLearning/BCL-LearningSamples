Video Tutorial

You may need to replace an existing video in your Video Cloud account for various reasons, such as:

  • You re-edit the video
  • You shoot a new version of the video to replace the old one
  • You set up a video as a remote asset, but now wish to store it on Video Cloud
  • You set up a video for a live streaming broadcast, but now that the event has passed, want to replace this with video-on-demand recording

Exercise: Bind video data in a template

In this exercise, you will learn how to bind video data to a template.

Examine video information

  1. Login to your Brightcove Video Cloud account.
  2. Navigate to the Media module.
  3. Select All Videos and locate the video that you uploaded in Exercise 1. If you have not uploaded a video yet, upload a video using Exercise 1 as a reference.
  4. With your video selected, click Edit. In the Edit Video dialog, you should see several fields that you can specify for your video, including Name, Short Description, Link URL, Thumbnail, and Reference ID.
  5. Browse to the the Flash-Only Player API documentation on the Brightcove Support site:
    • http://docs.brightcove.com/en/player/
  6. In the All Classes, select the MediaDTO class
  7. Review the Public Properties section. You should see properties that correspond to all of the video attributes, including displayName, shortDescription, linkURL, thumbnailURL, and referenceID
  8. Return to Video Cloud Studio.
  9. If you have not done so already, in the Edit Video dialog, fill out the Name, Short Description, Link Text, and Link URL fields.
  10. Select Save Changes.

Bind data to a video player

  1. Navigate to the Publishing module.
  2. Select All Templates and locate the BasicBEMLComponents template you created in Exercise 8.
  3. Select Edit.
  4. Locate the TitleLabel component and change the value of the text property so that it binds the videoPlayer.video.displayName value.

<TitleLabel text="{videoPlayer.video.displayName}"
   height="40"
   size="24"/>
  1. Locate the Label component and change the value of the text property so that it binds the videoPlayer.video.shortDescription value.
   
<Label text="{videoPlayer.video.shortDescription}"
   height="20"
   size="14"/>
  1. Locate the Link component and change the values of the text and url attributes so that they bind the following data objects:
    • text: videoPlayer.video.linkText
    • url: videoPlayer.video.linkURL
   
<Link text="{videoPlayer.video.linkText}"
   url="{videoPlayer.video.linkURL}"
   height="20"
   size="14"/>
  1. Select Save Changes.
  2. If you see a warning dialog that states, These changes may affect live players, select Save. Remember that you created a player from this template in a previous exercise.
  3. Navigate to the Media module and locate the video that you uploaded.
  4. In the panel on the right displaying your video's information, expand the Quick Video Publish option.
  5. In the Select a player drop-down list, select the BasicBEMLComponentsPlayer.
  6. Select the Preview this player button next to the drop-down list. You should see your video along with all of the data you specified for your video in the video player and its associated components. If you specified a working URL, you should be linked to the video's URL when you click the link text.
