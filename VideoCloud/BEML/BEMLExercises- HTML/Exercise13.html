Exercise: Explore the Media and Playlist DTOs

In this exercise, you will take a look at Brightcove's Flash-Only Player API documentation to examine the MediaDTO and PlaylistDTO classes.

Reference the Media and Playlist DTOs (Data Transfer Objects)

  1. Browse to the Brightcove Support site (support.brightcove.com).
  2. In the Developer References section of the page, select Flash-Only Player API Reference. You should be linked to the API documentation for the Flash-Only Player.
  3. In the All Classes section, select the MediaDTO class. This shows the various data transfer objects that are transferred down to the player when the player is loaded.
  4. Review the Subclasses field. Notice that the MediaDTO class contains two subclasses--AudioTrackDTO and VideoDTO. In this course, we are only concerned with VideoDTO subclass.
  5. In the Public Properties section, review the available DTOs.
  6. In the All Classes section, select the MediaCollectionDTO class and review the Subclasses field. Notice that the MediaCollectionDTO class contains two subclasses--AudioTrackPlaylistDTO and PlaylistDTO. In this course, we are only concerned with the PlaylistDTO class.
  7. Review the DTOs available for the class. In the next few, exercises you will learn how to use these DTOs to bind data to a video player.
