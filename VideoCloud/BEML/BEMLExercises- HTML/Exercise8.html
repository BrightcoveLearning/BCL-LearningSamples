Exercise: Use list components

In this exercise, you will learn how to use the List and TileList components.

Create a new video template

  1. Login to your Brightcove Video Cloud account.
  2. Navigate to the Publishing module.
  3. Select New Template.
  4. In the Create New Template dialog, name the template BEMLListComponents and paste the following starter code within the editor:
          
<Runtime>
   <Theme name="Deluxe" style="Light"/>
   <Layout id="application"
        boxType="vbox"
        width="995" height="700"
        padding="3">
        <Image width="480" height="80"
            source="http://img.brightcove.com/brightcove-logo.png"/>
        <Spacer height="10"/>
        <HBox>
            <VBox id="videoContainer"
                width="640" height="700">
                <ChromelessVideoPlayer id="videoPlayer"
                    width="640" height="450"/>
                <Spacer height="10"/>
                <TitleLabel text="My Video Title"
                    height="40"
                    size="24"/>
                <Label text="My video description"
                    height="20"
                    size="14"/>
                <Link text="My Link"
                    url="www.brightcove.com"
                    height="20"
                    size="14"/>
            </VBox>
            <Spacer width="10"/>
            <VBox id="listContainer"
                height="450">
            </VBox>
        </HBox>
   </Layout>
</Runtime>

Use the List and ListItem components

  1. Locate the listContainer VBox tag block nested within the HBox container.
  2. Within the listContainer VBox container, add a List component block with an id property set to videoList.
          
<VBox id="listContainer"
   height="450">
   <List id="videoList">
   </List>
</VBox>
  1. Within the List block, add a ListItem tag block with a boxType property set to vbox and a padding property set to 3.
          
<List id="videoList">
   <ListItem boxType="vbox"
       padding="3">
   </ListItem>
</List>
  1. Within the ListItem block, add a TitleLabel component and assign it the following properties and values:
    • text: My Video Title
    • size: 16
    • truncate: true
          
<List id="videoList">
   <ListItem boxType="vbox"
       padding="3">
       <TitleLabel text="My Video Title"
           height="20"
           size="16"
           truncate="true"/>
   </ListItem>
</List>
  1. Below the TitleLabel component, add a Label element and assign it the following properties and values:
    • text: My video description
    • height: 50
    • size: 14
    • truncate: true
          
<List id="videoList">
   <ListItem boxType="vbox"
       padding="3">
       <TitleLabel text="My Video Title"
           height="20"
           size="16"
           truncate="true"/>
       <Label text="My video description"
           height="50"
           size="14"
           truncate="true"/>
   </ListItem>
</List>
  1. Preview the template. The image, video player, and labels on the left side of the template should look familiar, as these are the components you laid out in your template in Exercise 8. Notice that the list component with list items you added are displayed to the right of the video player.
  2. Select Close.

Use the TileList component

  1. Change the List component to a TileList component. Make sure you leave the videoList id property.
  2. To the opening TileList component tag, add the following attributes and values:
    • columnWidth: 250
    • rowHeight: 90
          

<TileList id="videoList"
   columnWidth="250" rowHeight="90">
   <ListItem boxType="vbox"
       padding="3">
       <TitleLabel text="My Video Title"
           height="20"
           size="16"
           truncate="true"/>
       <Label text="My video description"
           height="50"
           size="14"
           truncate="true"/>
   </ListItem>
</TileList>
  1. Preview the template. You should see that the List component has been replaced with a TileList component. Notice that there is a lot of extra space within each list item. In the next section, you will learn how to add a thumbnail icon to the list item.
  2. Select Close.

Use the ThumbnailButton component

  1. In the opening ListItem tag, change the boxType property value to hbox.
  2. Within the ListItem component, nest the TitleLabel and Label components within a VBox container with a width property of 135.
  3. Within the ListItem block, above the VBox container, add a ThumbnailButton component with a width property of 115 and a height property of 85.

<TileList id="videoList"
   columnWidth="250" rowHeight="90">
   <ListItem boxType="hbox"
       padding="3">
       <ThumbnailButton width="115"
           height="85"/>
       <VBox width="135">
           <TitleLabel text="My Video Title"
               height="20"
               size="16"
               truncate="true"/>
           <Label text="My video description"
               height="50"
               size="14"
               truncate="true"/>
       </VBox>
   </ListItem>
</TileList>
  1. Preview the template. You should see the ThumbnailButton component displayed in the list items, with the TitleLabel and Label controls positioned to the right of the thumbnail. Notice that there is a lot of space below the Label control text.
  2. Select Close.
  3. To the Label control, change the text property to My extended video description.

<Label text="My extended video description"
   height="50"
   size="14"
   truncate="true"/>
  1. Preview the template. You should see that the label text does not fit on one line.
  2. Select Close.
  3. To the Label control, add the multiline property with a value of true.

<Label text="My extended video description"
   height="50"
   size="14"
   truncate="true"
   multiline="true"/>
  1. Preview the template. You should see that the label text now uses two lines.
  2. Select Close.
  3. Select Create Template.
