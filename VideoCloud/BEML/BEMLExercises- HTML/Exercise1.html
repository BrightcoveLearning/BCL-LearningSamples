Exercise: Upload videos and create a playlist

In this exercise, you will learn how to use Brightcove's Video Cloud Studio to upload videos and create a playlist for your videos.

Upload videos to your account

  1. Sign into your Brightcove Video Cloud Studio account.
  2. Navigate to Video Cloud Studio's Media module.
  3. Select the Add Videos button.
  4. In the Select file(s) to upload dialog, browse to the location of your video and select Open. You should see your video appear in the Standard Upload to Media Library dialog.

Note: If you want to upload more videos or upload multiple videos at once, select the Add More Files button.

  1. In the Standard Upload to Media Library dialog, change the Display Name to Test Video.
  2. In the Details column, add values for any required fields that you have defined.

Note: You need to fill in values for all required fields for the video to become active.

  1. Select the Upload All button. You should be transferred to your Upload queue, where you can monitor the status of your video upload.
  2. Once your video has finished uploading, in the Media Library section, select All Videos to view your video in your library.

Create a playlist for your videos

  1. In the Media module, select New Playlist.
  2. In the Create Playlist dialog, select Manual and name your playlist Test Playlist. Select Create Playlist.
  3. In the Playlist section, select All Playlists to view all of your account's playlists and double-click Test Playlist.
  4. In the Test Playlist window, select the Add Videos button.

Note: Do not select the Add Videos button with the green arrow in the left hand panel. This is the button that you used to add videos to your Video Cloud Studio account. Make sure you select the Add Videos button with the film strip icon in the playlist window.

  1. In the Add Videos To: Test Playlist dialog, select the video(s) that you uploaded and select Save & Close. You should see your video(s) added to the playlist.
