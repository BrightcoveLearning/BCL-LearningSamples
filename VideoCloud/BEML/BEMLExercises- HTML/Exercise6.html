Customize the ChromelessVideoPlayer component

In this exercise, you will learn how to customize the different elements of the ChromelessVideoPlayer component.

BEML is an HTML/XML language which is used to create a custom template.

This is the objectives for the document:

  1. Create a new video template
  2. Use the ChromelessControls and ToggleButton components
  3. Use the GraphicBlock component
  4. Use the Playhead component
  5. Add social media, fullscreen, and volume controls

Exercise: Customize the ChromelessVideoPlayer component

In this exercise, you will learn how to customize the different elements of the ChromelessVideoPlayer component.

Create a new video template

  1. Login to your Brightcove Video Cloud account.
  2. Navigate to the Publishing module.
  3. Select New Template.
  4. In the Create New Template dialog, name the template ChromelessBEMLTemplate.
  5. In the template editor, replace the existing code with the following code block:
          
<Runtime>
   <Theme name="Deluxe" style="Light"/>
   <Layout width="640" height="450">
        <ChromelessVideoPlayer id="videoPlayer"/>
        <!--<Canvas height="120" padding="10"
             includeInLayout="{!videoPlayer.fullscreen}">
             <GraphicBlock/>
             <VBox height="100" padding="10" vAlign="middle">
                  <Spacer height="10"/>
                  <TileList id="videoList" height="100" selectOnClick="true"
                       automaticAdvance="true" columnWidth="80"
                       rowHeight="100" columnGutter="3" buttonSize="26"
                       buttonOffsetY="-5">
                       <ListItem showBack="false" boxType="vbox">
                            <ThumbnailButton height="60" imageInset="3"
                                 data="{currentItem}"
                                 source="{currentItem.thumbnailURL}"/>
                            <Spacer height="4"/>
                            <TitleLabel text="{currentItem.displayName}"
                                 size="10" truncate="true"
                                 multiline="true"/>
                       </ListItem>
                  </TileList>
             </VBox>
        </Canvas>-->
  </Layout>
</Runtime>
  1. Review the code. Notice that the code contains a ChromelessVideoPlayer component and a Canvas container that is commented out. You will learn more about the commented code block later in this exercise.
  2. Select Preview Template. You should see a chromeless video player displayed. Notice that the player's controls are positioned at the bottom of the player by default.

Use the ChromelessControls and ToggleButton components

  1. Change the ChromelessVideoPlayer component tag to use tag block syntax.
          
<ChromelessVideoPlayer id="videoPlayer">
</ChromelessVideoPlayer>
  1. Within the ChromelessVideoPlayer tag block, nest a ChromelessControls tag block with a boxType property set to vbox.
          
<ChromelessVideoPlayer id="videoPlayer">
   <ChromelessControls boxType="vbox">
   </ChromelessControls>
</ChromelessVideoPlayer>
  1. Within the ChromelessControls block, add an HBox container with an hAlign property of center and a height of 60.
          
<ChromelessControls boxType="vbox">
   <HBox hAlign="center"
       height="60">
   </HBox>
</ChromelessControls>
  1. Within the HBox container, nest another HBox container with an id property of defaultView, a gutter property of 6, padding property of 10, and a vAlign property of middle.
          
<ChromelessControls boxType="vbox">
   <HBox hAlign="center"
       height="60">
           <HBox id="defaultView"
               gutter="6"
               padding="10"
               vAlign="middle">
           </HBox>
   </HBox>
</ChromelessControls>
  1. Within the HBox container, add a ToggleButton component with an id property of playButton, a width property of 65, and an iconName of play.
          
<HBox id="defaultView" ...>
   <ToggleButton id="playButton"
       width="65"
       iconName="play"/>
</HBox>

Note: You need to add the iconName property with the value play in order to display the play symbol in the button. Without the iconName property, the button would just be an empty rectangle.

  1. Preview the template. Notice that the play button is at the top of the video player, but it is currently covered by the overlay menu.
  2. Select Close.
  3. To the opening ChromelessVideoPlayer tag, add the useOverlayMenu property with a value of false.
          
<ChromelessVideoPlayer id="videoPlayer"
   useOverlayMenu="false">
   ...
</ChromelessVideoPlayer/>
  1. Preview the template. You should see that the overlay menu is no longer visible.
  2. Select Close.

Use the GraphicBlock component

  1. Below the ToggleButton component, add a Canvas container block with a height of 30.
          
<HBox id="defaultView" ...>
   <ToggleButton id="playButton".../>
   <Canvas height="30">
   </Canvas>
</HBox>
  1. Within the Canvas block, add an instance of the GraphicBlock element.
          
<Canvas height="30">
   <GraphicBlock/>
</Canvas>
  1. Preview the application. Notice a large rectangle has been added next to the play button. The GraphicBlock element is a non-interactive graphic that can be used as a skin behind other elements.
  2. Select Close.
  3. Below the GraphicBlock element, add an HBox container with a gutter property of 4 and a vAlign property of middle.
          
<Canvas height="30">
   <GraphicBlock/>
   <HBox gutter="4"
        vAlign="middle">
   </HBox>
</Canvas>
  1. Within the HBox container, add a Spacer component with a width property of 1.
          
<HBox gutter="4"
   vAlign="middle">
   <Spacer width="1"/>
</HBox>
  1. Below the Spacer element, add a Canvas container with a width of 40.
          
<HBox gutter="4"
   vAlign="middle">
   <Spacer width="1"/>
   <Canvas width="40">
   </Canvas>
</HBox>
  1. Within the Canvas container, add a Label component with an id property of positionLabel, a text property with a value of 0:00, a vAlign property of middle, an hAlign property of left, and an alpha property of .75.
          
<Canvas width="40">
   <Label id="positionLabel"
       text="0:00"
       vAlign="middle" hAlign="left"
       alpha=".75"/>
</Canvas>
  1. Preview the template. You should see the label added on top of the GraphicBlock component.
  2. Select Close.
  3. Copy the Canvas container with the nested positionLabel Label control, paste the copy below the container, and change the id property of the copied Label control to durationLabel.
          
<HBox gutter="4"
   vAlign="middle">
   <Spacer width="1"/>
   <Canvas width="40">
       <Label id="positionLabel"
           text="0:00"
           vAlign="middle" hAlign="left"
           alpha=".75"/>
   </Canvas>
   <Canvas width="40">
       <Label id="durationLabel"
           text="0:00"
           vAlign="middle" hAlign="left"
           alpha=".75"/>
   </Canvas>
</HBox>
  1. Preview the template. You should see the second label added right next to the first.
  2. Select Close.

Use the Playhead component

  1. Between the two Canvas containers, add a Playhead component with an id of playhead and a height property of 6.
          
<HBox gutter="4"
   vAlign="middle">
   <Spacer width="1"/>
   <Canvas width="40">
       <Label id="positionLabel".../>
   </Canvas>
   <Playhead id="playhead"
       height="6"/>
   <Canvas width="40">
       <Label id="durationLabel" .../>
   </Canvas>
</HBox>
  1. Preview the template. You should see the playhead added between the two labels. Notice that the space between the first label and the left edge of the Playhead component is larger than the space b/w the second label and the right edge of the Playhead component.
  2. Select Close.
  3. Between the Playhead component and the Canvas container nesting the durationLabel Label component, add a Spacer component with a width of 10.
          
<HBox gutter="4"
   vAlign="middle">
   <Spacer width="1"/>
   <Canvas width="40">
       <Label id="positionLabel".../>
   </Canvas>
   <Playhead id="playhead"
       height="6"/>
   <Spacer width="10"/>
   <Canvas width="40">
       <Label id="durationLabel" .../>
   </Canvas>
</HBox>
  1. Preview the template. You should now see a bigger space between the second label and the right edge of the playhead.
  2. Select Close.

Add social media, fullscreen, and volume controls

  1. Below the closing tag for the outermost Canvas container, add a Button component with an id of shareButton, a width property of 35, a height property of 30, and an iconName of socialShare.
          
<ChromelessVideoPlayer id="videoPlayer ...>
   <ChromelessControls ...>
       <HBox ...>
           <HBox ...>
               <ToggleButton .../>
               <Canvas ...>
                   ...
               </Canvas>
               <Button id="shareButton"
                   width="35" height="30"
                   iconName="socialShare"/>
           </HBox>
       </HBox>
   </ChromelessControls>
</ChromelessVideoPlayer>
  1. Preview the template. Notice the share button is added to the right of the playhead bar.
  2. Select Close.
  3. Below the shareButton component, add a ToggleButton component with an id of fullscreenButton, a width of 35, a height of 30, and an iconName of maximize.
          
<Button id="shareButton" .../>
<ToggleButton id="fullscreeButton"
   width="35" height="30"
   iconName="maximize"/>
  1. Below the ToggleButton component, add a VolumeControl component with an id property of volumeButton, a width of 35, a height of 30, and an iconName property of volume.
             
<Button id="shareButton" .../>
<ToggleButton id="fullscreeButton" .../>
<VolumeControl id="volumeButton"
   width="35" height="30"
   iconName="volume"/>
  1. Preview the template. Notice the fullscreen and volume buttons are added to the right of the share button.
  2. Select Close.
  3. Select Create Template to save the template.
  4. Select New Player.
  5. In the Create New Player dialog, name the player ChromelessBEMLPlayer and choose ChromlessBEMLTemplate as the template.
  6. Select Create Player.
  7. With the ChromelessBEMLPlayer selected, select Preview to preview the player in the browser. You should see the fullscreen and volume control buttons added to your player. Notice when you try to toggle the buttons, nothing happens. Later in this training series you will learn how to bind data to your player.
