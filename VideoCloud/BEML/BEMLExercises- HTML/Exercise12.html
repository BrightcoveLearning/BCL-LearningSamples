Exercise: Bind data using component id attributes

In this exercise, you will learn the importance specifying id attributes for some components.

Use the id attribute

  1. Login to your Brightcove Video Cloud account.
  2. Navigate to the Publishing module.
  3. Select All Templates and locate the BEMLListComponents template.
  4. Select Edit and review the code. You have learned that you can add the id attribute to all components, but using this attribute is optional unless you want to bind data. Notice that the ChromelessVideoPlayer, TabBar, and TileList components all contain id attributes of videoPlayer, playlistTabs, and videoList, respectively. These id values are important for data binding as you will see in the next few exercises.
