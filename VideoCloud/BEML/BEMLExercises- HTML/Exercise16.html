Exercise: Bind data for a list's selected item

In this exercise, you will learn how to bind data to a video player template using a list component's selected item.

Use the selectedItem and currentItem data objects

  1. Login to your Brightcove Video Cloud account.
  2. Navigate to the Publishing module.
  3. Select All Templates and locate the BEMLListComponents template.
  4. Select the Edit icon.
  5. Locate the ChromelessVideoPlayer component tag and add the video attribute with a value that binds the videoList.selectedItem property.
   
<ChromelessVideoPlayer id="videoPlayer"
   width="640" height="450"
   video="{videoList.selectedItem}"/>
  1. Locate the TitleLabel component and change the value of the text property so that it binds the videoPlayer.video.displayName value.
   
<TitleLabel text="{videoPlayer.video.displayName}"
   height="40"
   size="24"/>
  1. Locate the Label component and change the value of the text property so that it binds the videoPlayer.video.shortDescription value.
   
<Label text="{videoPlayer.video.shortDescription}"
   height="20"
   size="14"/>
  1. Locate the Link component and change the values of the text and url attributes so that they bind the following data objects:
    • text: videoPlayer.video.linkText
    • url: videoPlayer.video.linkURL
   
<Link text="{videoPlayer.video.linkText}"
   url="{videoPlayer.video.linkURL}"
   height="20"
   size="14"/>
  1. Locate the ThumbnailButton component within the ListItem container and add the source property with a value that binds the currentItem.thumbnailURL property.
   
<ThumbnailButton width="115" height="85"
   source="{currentItem.thumbnailURL}"/>
  1. Locate the TitleLabel component within the ListItem component and change the value of the text property to bind the currentItem.displayName property.
   
<TitleLabel text="{currentItem.displayName}"
   height="20"
   size="16"
   truncate="true"/>
  1. Locate the Label component within the ListItem component and change the value of the text property to bind the currentItem.shortDescription property.
   
<Label text="{currentItem.shortDescription}"
   height="50"
   size="14"
   truncate="true"
   multiline="true"/>
  1. Select Save Changes.
  2. If you see a warning dialog that states, These changes may affect live players, select Save. Remember that you created a player from this template in a previous exercise.
  3. Select All Players and locate the BEMLListComponentsPlayer you created in Exercise 15.
  4. Select Preview to view the player in the browser. You should see the videos in your playlist displayed in the TileList component and the first video in your playlist should be loaded into the video player. You should also notice that the video data you specified is displayed below the video as well. If you have more than one video in your playlist, you should be able to select another video from the list and play the video in the player.
