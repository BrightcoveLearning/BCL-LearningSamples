Exercise: Style video players

In this exercise, you will learn how to style a video player.

Use the Theme element to style a video player

  1. Login to your Brightcove Video Cloud account.
  2. Navigate to the Publishing module.
  3. Select All Templates and locate the BEMLListComponents template.
  4. Select Edit.
  5. Locate the Theme element and change the value of the style attribute to Dark.
   
<Theme name="Deluxe" style="Dark"/>
  1. Select Preview Template. You should see that your template now uses a darker style
  2. Select Close.
  3. To the Theme element, change the value of the name property to Minimal.
   
<Theme name="Minimal" style="Dark"/>
  1. Preview the template. You should notice that the style of the buttons and tabs have changed.
  2. Select Close.
  3. To the Theme element, change the value of the name property to Flat.
   
<Theme name="Flat" style="Dark"/>
  1. Preview the template. You should see that the buttons and tabs look like they have been flattened.
  2. Select Close.
  3. To the Theme element, change the value of the name property back to Deluxe.
   
<Theme name="Deluxe" style="Dark"/>

Use the Style component

  1. Change the Theme element so that it uses tag block syntax.

<Theme name="Deluxe" style="Dark>
</Theme>
  1. Within the Theme tag block, create a Style component block with an id property of videoList.
  2. Within the Style block, add a CDATA block.
   
<Theme name="Flat" style="Dark>
   <Style id="videoList">
       <![CDATA[ 
       ]]>
   </Style>
</Theme>
  1. Within the CDATA block, add the titleText selector with the following properties and values:
    • selected: #006699 (blue)
    • rollover: #000000 (black)

<Style id="videoList">
   <![CDATA[
       .titleText
       {
           selected: #006699;
           rollover: #000000;
       }
   ]]>
</Style>
  1. Below the titleText selector, add the listItemBoxes selector and assign it the following properties and values:
    • gradientTop: #333333 (dark gray)
    • gradientMiddle: #666666 (gray)
    • gradientBottom: #999999 (light gray)

<Style id="videoList">
   <![CDATA[
       .titleText
       {
           selected: #006699;
           rollover: #000000;
       }
       .listItemBoxes
       {
           gradientTop: #333333;
           gradientMiddle: #666666;
           gradientBottom: #999999;
       }
   ]]>
</Style>
  1. Preview the template. You should see the styles you added applied to the list component. Notice that the selected video's title text is blue. Also notice the gradient added to the unselected list items
  2. Select Close.
  3. Below the Style block, add another Style component block and assign it the class property with a value of TabBar.
  4. Within the Style block, add a CDATA block.

<Theme name="Flat" style="Dark>
   <Style id="videoList">
       <![CDATA[
           ...
       ]]>
   </Style>
   <Style class="TabBar">
       <![CDATA[
       ]]>
   </Style>
</Theme>
  1. Within the CDATA block, add the tabs selector and assign it the following properties and values:
    • gradientTop: #666666 (gray)
    • gradientBottom: #999999 (light gray)
    • selectedGradientTop: #333333 (dark gray)
    • selectedGradientBottom: #666666 (gray)
    • text: #006699 (blue)
    • selectedText: #FFFFFF (white)

<Style class="TabBar">
   <![CDATA[
       .tabs
       {
           gradientTop: #666666;
           gradientBottom: #999999;
           selectedGradientTop: #333333;
           selectedGradientBottom: #666666;
           text: #006699;
           selectedText: #FFFFFF;
       }
   ]]>
</Style>
  1. Preview the template. You should see your styles applied to the TabBar component. Notice that you have created two different gradients for the selected and unselected tabs.
  2. Select Close.
  3. Select Save Changes.
  4. If you see a warning dialog that states, These changes may affect live players, select Save. Remember that you created a player from this template in a previous exercise.
