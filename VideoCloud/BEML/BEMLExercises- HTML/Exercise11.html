Exercise: Use the banner components

In this exercise, you will learn how to use the two BEML banner components--Banner and ExpandingBanner.

Use the banner components

  1. Login to your Brightcove Video Cloud account.
  2. Navigate to the Publishing module.
  3. Select All Templates and locate the BEMLListComponents template you created in Exercise 9.
  4. Select Duplicate.
  5. With the BEMLListComponents copy template selected, click Edit.
  6. In the Edit Template dialog, rename the template BEMLBannerComponents.
  7. Locate the Image component and replace it with a Banner component. To the Banner component assign the following attributes and values:
    • width: 640
    • height: 80
    • source: http://img.brightcove.com/brightcove-logo.png
    • url: www.brightcove.com

<Runtime>
   <Theme .../>
   <Layout id="application" .../>
   <Banner width="640" height="80"
       source="http://img.brightcove.com/brightcove-logo.png"
       url="www.brightcove.com"/>
   ...
</Runtime>
  1. Select Preview Template. You should see the banner displayed above the video player.
  2. Select Close.

Use the ExpandingBanner component

  1. Remove the Banner component from the code and replace it with an ExpandingBanner component.

<Runtime>
   <Theme name="Deluxe"
       style="Light"/>
   <Layout id="application"
       boxType="vbox"
       width="995" height="700"
       padding="3">
   <ExpandingBanner/>
   <Spacer height="10"/>
   <HBox>
   ...
</Runtime>
  1. To the ExpandingBanner component, assign the following properties and values:
    • id: banner
    • width: 640
    • height: 80
       
<ExpandingBanner id="banner"
   width="640" height="80"/>
You should note that these width and height values only define the width and height of the banner when it is not expanded.
  1. To prevent users from selecting a video from the list component when the ad is showing, to the ExpandingBanner component, add the disableOnExpand attribute with a value of videoList.
   
<ExpandingBanner id="banner"
   width="640" height="80"
   disableOnExpand="videoList"/>
  1. To the ExpandingBanner component, add the following attributes and values that define the size and position of the banner when it is expanded:
    • expandedHeight: 406
    • expandDirection: down
    • expandedOffsetY: 10
   
<ExpandingBanner id="banner"
   width="640" height="80"
   disableOnExpand="videoList"
   expandedHeight="406"
   expandDirection="down"
   expandedOffsetY="10"/>
  1. Select Save Changes.
  2. If you see a warning dialog that states, These changes may affect live players, select Save. Remember that you created a player from this template in a previous exercise.
  3. Select New Player.
  4. In the Create New Player dialog, name the player BEMLBannerPlayer and select the BEMLBannerComponents template.
  5. Select Create Player.
  6. With the BEMLBannerPlayer selected, click Preview to preview the player in the browser. You should see...
  7. Navigate to the Advertising module.
