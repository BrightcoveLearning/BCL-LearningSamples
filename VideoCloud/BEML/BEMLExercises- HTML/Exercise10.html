Exercise: Use the ViewStack component

In this exercise, you will learn how to use the ViewStack component.

Create a new video player template

  1. Login to your Brightcove Video Cloud account.
  2. Navigate to the Publishing module.
  3. Select New Template.
  4. In the Create New Template dialog, name the template BEMLViewStack and replace the existing code in the editor with the following starter code:

  1. Review the code. You should see that the code defines a chromeless video player and a ViewStack container with three different views, each laying out the player's controls out in different ways.

Use the ViewStack container

  1. Select Preview Template. You should see a chromeless video player displayed with its chromeless controls, including a play button, playhead, and volume button.
  2. Select Close.
  3. Locate the HBox container for the view1 instance nested in the ViewStack container and review the code. You should see that this container defines the video player and controls you saw when you previewed the template. You have learned that the ViewStack container is a layout container that only makes a single child visible at a time. To display other views in the stack you must define the selectedIndex property in the ViewStack container.
  4. To the opening ViewStack container tag, add the selectedIndex attribute with a value of 0.

<ViewStack selectedIndex="0">
    <HBox id="view1"
          hAlign="center"
          height="60"
          y="380">
   ...
</ViewStack>
  1. Preview the template. You should see the same player and controls displayed. This occurs because the selectedIndex property is set to 0, the first view in the stack, by default.
  2. Select Close.
  3. To the ViewStack container, change the value of the selectedIndex property to 1.
  4. Preview the template. You should see that the player's controls have moved to the top of the player.
  5. Select Close.
  6. To the ViewStack container, change the value of the selectedIndex property to 2.
  7. Preview the template. You should see that the player's controls are back at the bottom of the page. However, you should notice that the playhead is not displayed. This type of player is common for playing video ads or live video streams.
  8. Select Close.
  9. Select Create Template.
