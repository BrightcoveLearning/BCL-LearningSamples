Exercise: Use the ComboBox and TabBar components

In this exercise, you will learn how to apply the ComboBox and TabBar components to a video player.

Use the ComboBox component

  1. Login to your Brightcove Video Cloud account.
  2. Navigate to the Publishing module.
  3. Select All Templates and locate the BEMLListComponents template you created in Exercise 8.
  4. Select Edit.
  5. In the editor, above the TileList component within the VBox container, add a ComboBox component and assign it the following properties and values:
    • id: playlistCombo
    • width: 355
    • height: 30
    • label: Select a playlist...
          
<VBox height="450">
   <ComboBox id="playlistCombo"
       width="355" height="30"
       label="Select a playlist..."/>
   <TileList id="videoList"
       columnWidth="250" rowHeight="90">
       <ListItem boxType="vbox"
           padding="3">
           <TitleLabel text="My Video Title"
               height="20"
               size="16"/>
           <Label text="My video description"
               height="20"
               size="14"
               truncate="true"
               multiline="true"/>
       </ListItem>
    </TileList>
</VBox>
  1. Preview the template. You should see the ComboBox component added above the TileList component. In a later exercise, you will learn how to bind data to components, such as the ComboBox.
  2. Select Close.

Use the TabBar component

  1. Remove the ComboBox component and replace it with a TabBar element.
  2. To the TabBar element, add the id property with a value of playlistTabs and a height property of 30.

<VBox height="450">
  <TabBar id="playlistTabs" height="30"/>
  <TileList id="videoList"
      columnWidth="250" rowHeight="90">
      ...
   </TileList>
</VBox>
  1. Preview the template. You should see that the ComboBox component has been replaced with the TabBar component.
  2. Select Close.
  3. Select Save Changes.
  4. If you see a warning dialog that states, These changes may affect live players, select Save. Remember that you created a player from this template in a previous exercise.
