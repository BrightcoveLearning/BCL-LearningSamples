Brightcove Player 6 Now Default Version

Brightcove Player 6 is now the default version for newly created players, a major version update.

A change to the major version indicates a significant change to the way the API works, and a few of the changes Brightcove made to simplify the API could break some integration code. Because of this, and in order to prevent problems with your site, automatic updates will not be made for major version updates. That means that your player will effectively be locked at the final release of Brightcove Player version 5, even if you have selected automatic update mode. This means you will need to manually update to Brightcove Player 6 once your testing is complete.

For more information on Brightcove Player 6 see: Brightcove Player 6 Update

For information on controlling versions see: Setting the Player Update Mode and Version
