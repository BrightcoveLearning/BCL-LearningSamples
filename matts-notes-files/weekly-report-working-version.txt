Title: Static Image Overlays
Description: You can use the static image overlay feature to superimpose a static image onto a video in a Cloud Playout channel. A static image is a still image that doesn't have motion. You prepare the image and store it outside of Cloud Playout and provide to us as a publicly accessible , unauthenticated http/https URL.
Link: https://studio.support.brightcove.com/cloud-playout/static-image-overlays.html
Category: Cloud Playout
Status: Released
