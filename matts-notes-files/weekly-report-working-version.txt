Title: Scheduling Activation/Deactivation of an SEP Stream
Description: With the JobStartStop endpoint, you can schedule the activation and deactivation of an existing live SEP job. At the specified time, our system will make the necessary API calls to activate and de-active a Job on your behalf.
Link: https://apis.support.brightcove.com/live-api/guides/live-scheduler-stop-start.html
Category: Live API
Status: Released

Title: Scheduling Clip Generation for an Live Streams
Description: With the Clip endpoint, you can schedule the clip event of an existing live job. At the specified time, our system will make the necessary API calls to start a Clip Job on your behalf.
Link: https://apis.support.brightcove.com/live-api/guides/live-scheduler-clips.html
Category: Live API
Status: Released
