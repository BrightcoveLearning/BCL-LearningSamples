Title: Live DVRUX Plugin
Description: In Brightcove Player versions previous to 6.33.0 the Live DVRUX plugin was required to play live streams. This is no longer the case, as that functionality is now built into Brightcove Player. See the Playing Live Streams document for further details. The Live DVRUX plugin is now deprecated and should no longer be used.
Link: https://support.brightcove.com/live-dvrux-plugin
Category: Brightcove Player
